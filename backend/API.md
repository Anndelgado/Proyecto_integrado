# API Barranquilla Convive

## Users

### GET /users
obtener todos los usuarios.

### GET /users/:id
obtener usuario especifico por id.

### POST /users
crear un nuevo usuario.

### PUT /users/:id
actualizar un usuario especifico por id.

### DELETE /users/:id
eleminar un usuario especifico.


## Institutions

### GET /institutions/:id

### POST /institutions

### PUT /institutions:id

### DELETE /institutions/:id
